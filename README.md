# Graph-Colouring
Given a graph G(V,E), we can define χ(G), the chromatic number of the graph, as the minimum number of colours needed to colour the graph such that no two adjacent nodes are of the same colour. Consider the problem of identifying a colouring of the graph such that the colour(x) 6 = colour (y) if there exists an edge (x,y) in the graph.
